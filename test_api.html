Troubleshooting Guide

Common Issues:
  • All APIs failing: Make sure XAMPP Apache and MySQL are running
  • Database connection failed: Run database/setup_database.php first
  • 404 errors: Check that the project is in the correct XAMPP htdocs folder
  • CORS errors: Make sure .htaccess file exists in the project root
